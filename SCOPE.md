# Scope — hestiaos-templates

## In scope

```
- Repo skeletons (green-skeleton)
- Document templates (ADR, backlog item, runbook, evidence record, provenance note, security review, agent task)
- Governance templates
- Agent-task templates
- Evidence / provenance templates
```

## Out of scope

```
- Product code / source
- Secrets, tokens, keys, .env
- Infrastructure configs
- Customer / personal data
- Runtime artifacts
- Old git history / source-copy from legacy repos
```
